Introduction

This report was originally designed to offer professional-level accounting data for businesses who use Zen Cart heavily, with multiple display options, several levels of data output and flexible search options. It was tested in a Zen Cart shop with 116,000 orders. If it works for them, it should work for you.

Its features are explained in the report's wiki, but the report has been built with usability in mind, even for the most simplistic of users. Using the report should be fairly obvious and self-explanatory, but please let me know if you feel something can be tweaked/added/removed to offer a more user-friendly experience.

Prerequisites

  • A Zen Cart installation, v1.5.8 or later.
  • PHP 7.3 through 8.2
  • Javascript must be enabled in your browser.

Installation

Starting with v4.0.0, the report is released as a Zen Cart encapsulated plugin, so installation (and removal) is simplified.

  1. Merge the plugin's /zc_plugins with your site's pre-existing /zc_plugins directory.
  2. Log into your Zen Cart admin panel.
  3. Use the admin menus, navigating to Modules :: Plugin Manager.
  4. You'll see a plugin named Sales Report II; click the Install button for the plugin. That's it!

Usage

Once you've installed the report, you will see an addition to the admin's Reports main menu tab named Sales Report II; click that menu item and start generating reports.

Uninstall

To uninstall the report, navigate to the admin's Modules :: Plugin Manager, find the line-item for Sales Report II and click the Un-Install button.
